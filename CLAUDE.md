# CLAUDE.md - Cavele Digital Website Project

## Project Overview
This is the codebase for caveledigital.co.za, the marketing website for Cavele Digital - a solo web development / digital solutions business run by Joseph Cavele, based in Kroondal Ikemeleng, Rustenburg, North West, South Africa.

Stack: Node.js / Express / MongoDB backend, React / Next.js frontend.

## Core Philosophy
People buy honest. This site previously contained fabricated team members, fake founding history, and invented statistics - all of which have been identified and must never return. The entire positioning of this site is built on one idea: real proof from a real solo developer converts better than fabricated scale.

## Site Structure Decision
- One page - no separate pricing page or pricing section. Pricing is handled via direct quote after contact ("Get a Quote" CTA).
- Primary CTA across the entire site: "Get a Quote"
- Sections, top to bottom: Hero -> Services -> Portfolio -> About -> Why Choose Us -> 3-Step Process -> FAQ -> Contact/Final CTA -> Footer
